# FRAMC Docs

The official documentation for the FragMC plugin ecosystem, including **IcedSpear**, **WebLink**, and more.

## Local Development

To run this documentation site locally:

1. **Install Dependencies**

   ```bash
   npm install
   ```

2. **Start Development Server**

   ```bash
   npm run docs:dev
   ```

3. **Build for Production**
   ```bash
   npm run docs:build
   ```
